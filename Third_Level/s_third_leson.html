طباعة سطر جديد:

لطباعة جملتين أو أكثر كل جملة بسطر منفرد نستخدم إحدى الطرقتين التاليتين :
  • 1-endl
  • 2-"n\"

    • 1-endl :

    main.cpp
    
#include <iostream>
using namespace std;
int main() {
  cout<<"hello student"<<endl<<" you are in LearnNow";
    return 0;
}
    Output:
      hello student
  you are in LearnNow
    ويمكن كتابتها بعدة طرق:
    main.cpp
    
#include <iostream>
using namespace std;
int main() {
  cout<<"hello student";
  cout<<endl<<" you are in LearnNow";
    return 0;
}
    Output:
      hello student
  you are in LearnNow
    main.cpp
    
#include <iostream>
using namespace std;
int main() {
  cout<<"hello student";
  cout<<endl;
  cout<<" you are in LearnNow";
    return 0;
}
    Output:
      hello student
  you are in LearnNow

    2-"n\" :

    ويجب أن تكون داخل تنصيصتين " " أو تنصيصة ' '.
    main.cpp
    
#include <iostream>
using namespace std;
int main() {
  cout<<"hello student\n"<<" you are in LearnNow";
    return 0;
}
    Output:
      hello student
  you are in LearnNow
    main.cpp
    
#include <iostream>
using namespace std;
int main() {
  cout<<"hello student"<<'\n'<<" you are in LearnNow";
    return 0;
}
    Output:
      hello student
  you are in LearnNow
    و بنفس الطريقة السابقة يمكن كتابتها بعدة طرق:
    main.cpp
    
#include <iostream>
using namespace std;
int main() {
  cout<<"hello student";
  cout<<"\n"<<" you are in LearnNow";
    return 0;
}
    Output:
      hello student
  you are in LearnNow
    main.cpp
    
#include <iostream>
using namespace std;
int main() {
  cout<<"hello student";
  cout<<"\n";
  cout<<" you are in LearnNow";
    return 0;
}
    Output:
      hello student
  you are in LearnNow
    ويمكن طباعة أكثر من سطر
    main.cpp
    
#include <iostream>
using namespace std;
int main() {
  cout<<"hello student";
  cout<<"\n\n\n"<<" you are in LearnNow";
  cout<<endl<<endl<<endl<<" welcome";
    return 0;
}
    Output:
      hello student


  you are in LearnNow


  welcome

    لا تنسى الممارسة والتطبيق العملي لأنك لن تفهم إذا لم تطبق ماتعلمته ولا يغرك الفهم اللحظي وأنت تقرأ فكله سيتبخر بمرور الوقت إذا لن تدعمه عملياً .

    التعليقات (Comments):

    هو نص يتم إضافته إلى الشيفرة المصدرية (الكود) لشرحها أو توضيحها، ويتجاهله المترجم أو المفسر تماماً عند تنفيذ البرنامج. تُستخدم التعليقات لتسهيل فهم الكود على المبرمجين، سواءً لأنفسهم عند العودة إليه لاحقاً أو لغيرهم الذين يعملون معه. وهي لا تظهر في المنتج النهائي ولا تؤثر على أداء البرنامج.


    يتم تعليق سطر بكتابة شرطتين // أو تعليق أكثر من سطر باستخدام /* */ حيث أن المترجم لايقرأ مايكتب بعد // ولا مابين /* */
    main.cpp
    
#include <iostream>
using namespace std;
int main() {
  //بيانات الطالب
  cout<<"name: mohammed";
  cout<<"\n";
  cout<<"age: 23";
    return 0;
}
    Output:
      name: mohammed
  age: 23
    main.cpp
    
#include <iostream>
using namespace std;
int main() {
/*طباعة بيانات الموظف 
وبيانات الشركة التي يعمل فيها*/

  cout<<"phone number: 24734832\n";
  cout<<"salary: 4000\n";
  cout<<"company: MTN\n";
  cout<<"location: sana'a\n";

}

age: 23
