المتغيرات (variable)

هي مساحات في الذاكرة تُستخدم لتخزين البيانات التي يمكن أن تتغير قيمتها أثناء تشغيل البرنامج. كل متغير له اسم ونوع بيانات محددين، وتُستخدم المتغيرات لتخزين معلومات مثل الأرقام والنصوص والتواريخ وغيرها

طريقة انشاء متغير

;data_type varible_name= value

أنواع المتغيرات أو البيانات (data type):

  • int: تستخدم لتخزين الاعداد الصحيحة
    مثل: 1,2,3...
  • flout: تخزن الاعداد العشرية
    مثل : 2.3 , 0.5 ,5.6
  • double: تخزن القيم العشرية أيضًا لكنها قادرة على استيعاب أعداد أكثر بعد الفاصلة العشرية حيث أن float لا تستوعب إلا 7 أرقام فقط بعد الفاصلة العشرية!
  • string: تخزن النصوص كالاسماء والجمل والفقرات
  • char: تستخدم لتخزين الحروف
    مثلاً: a,b,c
  • boolean: وتخزن قيمتين فقط وهما : true أو false

قواعد تسمية المتغيرات:

يمكنك أن تسمي المتغير الذي ستستخدمه في برنامجك أي اسم تريد و الأفضل أن يدل الاسم على ماسيخزنه المتغير لكن عليك ان تلتزم بعدة قواعد وهي:
  • يجب أن يكون الاسم باللغة الإنجليزية , ...
  • أن يبدأ الاسم بحرف أو شرطة سفلية ( _ ) ولايبدأبرقم:

    اسم , متغير1 , 1x , 23name...

  • عدم إمكانية استخدام المسافات أو الأحرف الخاصة أو الكلمات المحجوزة:

    my name , my$name$ , cout , int...

  • ويسمح باستعمال الأحرف (الكبيرة والصغيرة) والأرقام والشرطة السفلية فقط

    x , y , a , name , first_var , var2 , name2

    return 0;

Output:

age: 24

#include <iostream>
using namespace std;
int main() {
  string name="Ali";
  cout<<name<<" is programmer\n";
  cout<<name<<" leaves in yemen\n";
  cout<<"the best programmer  is "<<name<<endl;
  cout<<"where is "<<name<<" now?";
    return 0;

Output:
Ali is programmer
Ali leaves in yemen
the best programmer  is Ali
where is Ali now?
Note:

لاحظ في المثال السابق انه في حالة عدم استخدام متغير وقمت بكتابة اسم الشخص مباشرة فإانك عندما تريد تغيير الاسم إلى مازن مثلاً فعليك الذهاب لكل سطر من الكود وتبديل الاسم القديم بالجديد وهذا أمر متعب خاصة إن كان الكود طويل ,لكن عند تخزين الاسم داخل متغير فإنك لن تضطر للذهاب لكل موضع ذكر فيه الاسم بل انك ستكتفي بتغيير قيمة المتغير فقط.

أول مشروع لك في ++C

main.cpp

#include <iostream>
using namespace std;
int main() {
  //طباعة بيانات الطلاب في مدرسة
  //المتغيرات التي ستخزن بيانات الطالب
  int age=21;
  string name="ahmed";
  char grade='B';
  double gpa=3.2;
  bool success=true;
  cout<<"your name is: "<<name<<endl;
  cout<<"your age is : "<<age<<endl;
  cout<<"your grade is : "<<grade<<endl;
  cout<<"your gpa is : "<<gpa<<endl;
  cout<<"success : "<<success;
    return 0;
}
Output:
your name is: ahmed
your age is : 21
your grade is : B
your gpa is : 3.2
success : 1
